# S1-07-T-BackEnd

BackEnd de Mejor Postor, PWA de subastas online.

## Stack/Tecnologías utilizadas

- NodeJS
- Express
- MongoDB
- BcryptJS
- JWT
- Nodemailer
- Mercadopago

- PORT_SERVER_FRONT = 3000
- URL_SERVER = "http://localhost"
- PORT_HEROKU = ""
- STRING_SERVER_MONGODB = ""
- EMAIL_SERVICE = ""
- EMAIL_USER = ""
- EMAIL_PASS = ""
- MP_TOKEN = ""
